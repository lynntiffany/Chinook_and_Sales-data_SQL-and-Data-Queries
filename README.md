# Chinook SQL Analysis

## Project Overview

This project demonstrates SQL querying and relational database analysis using the Chinook sample database. The objective was to strengthen SQL skills by exploring database relationships, extracting insights, and solving business-oriented analytical problems.

---

## Objectives

- Import the Chinook database into MySQL.
- Understand relational database design and schema relationships.
- Write core and advanced SQL queries.
- Answer business questions using SQL.
- Explore basic query optimization techniques.

---

## Database Schema

The Chinook database consists of interconnected tables, including:

- Artist
- Album
- Track
- Customer
- Invoice
- InvoiceLine
- Employee
- Genre
- MediaType
- Playlist
- PlaylistTrack

---

## SQL Concepts Covered

### Core SQL
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions (COUNT, SUM, AVG)

### Advanced SQL
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Subqueries
- Window Functions
  - ROW_NUMBER()
  - RANK()
  - PARTITION BY

### Business Analysis
- Top-performing products
- Top customers
- Revenue trends
- Customer purchasing behavior

### Query Optimization
- Indexing concepts
- EXPLAIN statement
